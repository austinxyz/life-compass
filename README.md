# 🧭 人生指南针 Life Compass

用设计思维梳理人生现状，探索多条未来路径，输出完整的人生规划文档。

## 快速开始

1. 在这个目录打开 Claude Code
2. 说："use life-compass skill"（或 `/life-compass`）
3. 跟着进度面板一步一步来

## 步骤概览

| 步骤 | 技能名 | 内容 | 预计时间 |
|------|--------|------|---------|
| 1 | lc-dashboard | 仪表盘：四个领域打分 | 15-20分钟 |
| 2 | lc-compass | 指南针：工作观与人生观 | 20-30分钟 |
| 3 | lc-journal | 活力日志：找到你的能量来源 | 20分钟 |
| 4 | lc-gravity | 松绑问题：重新框定卡点 | 15分钟 |
| 5 | lc-odyssey | 三条路径：五年方案设计 | 30-45分钟 |
| 6 | lc-synthesis | 综合规划：生成完整文档 | 自动完成 |

## 参考样例

`examples/austin/` 目录有完整填写的样例，可以参考格式和深度。

## 支持多人

家庭成员可以共用这个工具。每个人的答案独立保存在 `outputs/{你的名字}/`。

## 灵感来源

受《人生设计课》（Bill Burnett & Dave Evans，斯坦福 d.school）启发，内容独立原创。
